## 📊 Commit Message Conventions

Use these prefixes for professional commits:

- `feat:` - New feature
- `fix:` - Bug fix
- `docs:` - Documentation
- `test:` - Tests
- `chore:` - Maintenance
- `refactor:` - Code refactoring
- `style:` - Code style changes
- `perf:` - Performance improvements
